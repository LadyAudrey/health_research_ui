## Requirements

## Setup

1. Clone repo:

```bash
git clone --recurse-submodules https://github.com/LadyAudrey/health_research_ui.git
```

2. Navigate to the `health_research_ui` directory:

```bash
cd ./health_research_ui
```

3. Install dependencies:

```bash
npm install
```

4. Run the dev server:

```bash
npm run dev
```
